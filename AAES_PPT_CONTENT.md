## Slide 2: The Problem Statement
**Header:** Challenges in Modern Academic Evaluation
**Bullet Points:**

- Fragmented systems for attendance, mentorship, and assignments.

## Slide 3: Our Solution
**Header:** The AAES Ecosystem
**Bullet Points:**
- **Automated Grading:** OCR + LLM (Gemma 3) for theory; Sandboxed Runner for Code.
- **Role-Based Access:** Specialized workflows for all academic stakeholders.
- **RAG-Based AI Tutor:** Students can query study materials directly via AI.
- **Unified Governance:** Integrated CCM and Mentorship tracking.

---

## Slide 4: System Architecture
**Header:** Robust Tech Stack
**Bullet Points:**
- **Frontend:** React + TypeScript + Tailwind CSS (High-Performance UI).
- **Backend:** Node.js + Express + MongoDB (Scalable & Secure).
- **AI Service:** FastAPI + Ollama (Local AI Privacy) + Tesseract OCR.
- **Security:** JWT Authentication & Role-Based Authorization (RBAC).

## Slide 7: AI Engine Deep Dive
**Header:** Intelligence at Scale
**Bullet Points:**
- **Theory Evaluation:** LLM-driven rubric matching for descriptive answers.
- **Code Execution:** Safe, sandboxed execution of Python/C/Java against test cases.
- **Plagiarism Detection:** Intra-batch similarity checking using TF-IDF.
- **OCR Logic:** Robust text extraction from handwritten/scanned documents.
